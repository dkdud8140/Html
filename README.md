# vi editer 실습
* 파일 열기 : vi 파일이름
* Insert(편집) 모드 진입하기  
vi 처음화면에서 i
* OverWrite(평집) 모드 진입하기  
vi 처음 화면에서 o

* 편집화면에서 명령모드로 전환하기  
ESC 키( 1~2)
* vi 종료하기  
: q 
* 편집내용 저장하기  
:w
* 편집내용 저장하지 않고 끝내기  
:q!
* 편집한 내용 저장하면서 끝내기  
:wq
